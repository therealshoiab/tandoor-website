# Tandoor Restaurant Website

This is a premium, modern React + Vite + Tailwind CSS (v4) website for Tandoor Restaurant.

## Project Structure

```
/restaurant-website/
  ├── .agents/
  │   ├── workflows/
  │   │   └── build-restaurant-site.md
  │   └── skills/
  ├── src/
  │   ├── components/
  │   ├── pages/
  │   ├── assets/
  │   └── data/
  ├── public/
  ├── production_artifacts/
  └── README.md
```

## Stack and Libraries

- **Framework**: React + Vite (non-interactive scaffolded)
- **Styling**: Tailwind CSS v4
- **Routing**: `react-router-dom`
- **Animations**: `framer-motion`
- **Icons**: `lucide-react`
- **Notifications**: `react-hot-toast`
